**Meta Ads Performance Intelligence & Optimization Simulator
Overview**

This project simulates a Meta Ads growth intelligence system that analyzes campaign performance, calculates key marketing KPIs, and optimizes budget allocation using data-driven logic.
It is designed to replicate how modern growth teams and AI-driven marketing systems make decisions on ad performance, targeting, and spend efficiency.

**Problem Statement**
Marketing teams often struggle with:

Understanding which ads drive real ROI
Allocating budget efficiently across campaigns
Identifying high-performing audiences
Tracking performance across fragmented metrics

This project solves these issues by building a simplified AI-style optimization pipeline.

 System Workflow
Generate synthetic Meta Ads dataset
Compute performance KPIs:
CTR (Click Through Rate)
CPC (Cost Per Click)
CPA (Cost Per Acquisition)
ROAS (Return on Ad Spend)
Score each ad using performance signals
Automatically reallocate budget toward best-performing ads
Visualize campaign-level insights

Key Features
 Synthetic Meta Ads dataset (300+ ad records)
 KPI computation engine (CTR, CPC, CPA, ROAS)
 Budget optimization algorithm (score-based allocation)
 Campaign + audience performance analysis
 Visualization of performance trends
